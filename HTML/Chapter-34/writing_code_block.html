Writing code block in HTML using <pre> tag and <code> tag

    
    fruits = ["apple", "banana", "cherry"]
    for x in fruits:
        if x == "banana":
        break
        print(x)
    
       
        
            fruits = ["apple", "banana", "cherry"]

              if x == "banana":

Note: In both the above code snippets the indents that was provided in the text, is displayed as it is.

Without using <code> tag

    fruits = ["apple", "banana", "cherry"]
    for x in fruits:
        if x == "banana":
        break
        print(x)

Note: Both <pre> and <code> tag works the same. But to represent code snippet we usually use <code> tag. And using just code tag does not represent the code snippet as we want.

The below example shows the output when we use <code> tag without using <pre> tag.

fruits = ["apple", "banana", "cherry"] for x in fruits: if x == "banana": break print(x)
